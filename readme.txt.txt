Anleitung zur Nutzung von "Blender" zum Starten von Mines3D.


1. Installation von Blender:
	
	Das Programm Blender ist auf der offiziellen Internetseite von Blender kostenlos zum Download verfügbar.
	Die Installationsdatei ist unter follgendem Link zu finden: www.blender.org/download/.
	Um andere Versionen des Programmes zu finden, müssen unter dem "Download" Knopf andere Versionen durchsucht werden.
	Hier muss die zum System passenden Version Ausgewählt werden.

		Für Windows Versionen gillt: 

	Wurde eine Installer Version Ausgewählt muss diese geöfnet werden. Wenn die Installation vorbereitet ist mussd auf "next" gedrückt werden.
	Es muss als nächstes ein Harcken zur bestätigung der Lizensen gesetzt werden, bevor man fortfahren kann. 
	Jetzt wird der Speicherfahrt der Datei Ausgewählt. Standartweise ist dieser ein Ordner names "Blender Foundation" unter den "Programme" Ordner.
	Sobald dies Ausgewählt wurde kann Blender nun Installiert werden.
	NAchdem das Programm Installiert wurde muss auf "Finish" die Installation beendet werden.

	Wurde eine .zip Datei Ausgewählt, muss diese geöfnet werden. Der Inhalt der datei muss entpackt werden. Dies kann ein belibiger Ort sein.
	Die daraus entsehende Datei ist ein Ordner der Blender enthält.


2. Öffnen von Blender:
	
	Wenn die Variante "Installer" genutz wurde, ist eine verknüpfung des Programmes auf dem Desktop zu finden.
	Ist diese nicht vorhanden kann man die Blender.exe in dem Installationmsordner öffnen der zuvor erstellt wurde.
	Standartmässig ist dieser unter Programme -> BlenderFoundation -> Blender.
	Ist eine zip. Datei zur Installation benutzt worden kann man Die Blender.exe Datei in dem zuvor Entpackten Ordner finden.


3. Öffnen der Datei Mines3D:

	Das Prgramm Mines3D wird entweder Standartmässig mit Blender geöfnet.
	Fals nicht, muss mit Rechtsklick und dann "Öffnen mit" nach der Blender.ex oder ihrer Verknüpfung gesucht werden.
	Es muss nur bestätigt werden das Die dAtei mit jenem Programm geöfnet werden soll.


4. Orientierung in Blender:
	
	Um zwischen den verschiedenen Szenen des Soieles zu wechseln muss man am oberen Bildschirmrand drücken, wo steht:
		- Menue (Das Menü des SPieles)
		- Game (Das HAuptspiel)
		- Optionen (Einstellungen des Spieles)
		- Hilfe (Hilfeoptionen im Spiel)
	Um das Spiel in Vollbild zu speilen muss die TAstenkombination "STRG - ARROWUP" benutz werden.
	Zum Spielen der jeweiligen Szene muss "P" gedrückt werden.
	Empfolen wird, das spiel in der Szene Menü zu Starten, da von hier jede andere Szene erreicht werden kann.
	Um einen Überblick über die "Game-Logic" zu erhalten kann neben dem Reiter zur AUswahl der Szene der Reiter geöfnet werden, bei welchen STandartmässig Default steht.
	Die Game Logic findet man unter "Game Logic".
